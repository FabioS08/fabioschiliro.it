I develop my Master's Thesis, titled "Real-Time Multimodal Road Users Detection for Smart Infrastructures via Optimized Point Cloud Filtering".
The main focus of this project was on the deployment of multimodal 3D Object Detection models on edge computing platforms, specifically the NVIDIA Jetson AGX Orin. The research investigated optimized point cloud filtering strategies aimed at reducing computational overhead while preserving high detection accuracy.
The principal objective was to achieve an effective trade-off between robust detection performance and the stringent low-latency requirements of real-time Smart Infrastructure applications.
